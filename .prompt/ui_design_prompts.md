# UI 设计稿 Prompt（给墨刀 AI 使用）

> 以下 prompt 用于在墨刀 AI 等工具中生成初步设计稿，后续可反复调整优化。

---

## Prompt 1：学生门户 PC 端（student-pc）

```
生成一个学生门户网站的 PC 端页面，为学生提供校园信息查看服务，不是后台管理。

页面顶部为导航栏，蓝色 #1890ff 底色，白色文字。
导航菜单从左到右依次为：首页、我的课程、成绩查询、校园生活（含子菜单：食堂菜单、每日书籍、每日游戏）、个人中心。

首页布局：
- 上方为欢迎区域，左侧显示"欢迎回来，张三"和今天的日期，右侧显示一条校园公告或每日一句。
- 中间为 3 个快捷入口卡片横排：查看课表、最新成绩、今日菜品。每个卡片带图标和简短文字。
- 下方左侧为"本周课程安排"小型日历/列表，右侧为"最近公告"滚动列表。

整体风格清新明快，蓝色主调搭配白色和浅灰色。页面宽度 1200px 居中布局。
```

---

## Prompt 2：学生门户 - 成绩查询页

```
生成一个学生门户网站的成绩查询页面，PC 端。

页面顶部导航栏与首页保持一致，当前高亮"成绩查询"。
标题区显示"我的成绩"和当前学期选择器。

下方以学期为分组，每个学期为一个卡片区块，展示该学期的课程成绩列表表格：
表格列：课程名称、学分、成绩（数字，及格标绿、不及格标红）、绩点、考核方式（考试/考查）。
每门课行末显示排名标签（如：全班第 5/45）。

最下方汇总栏显示：学期总学分、平均绩点、学期排名。
右侧有一个悬浮的"成绩趋势"小卡片，用简单折线图占位展示各学期绩点变化。
```

---

## Prompt 3：学生门户 - 我的课程页

```
生成一个学生门户网站的课程查看页面，PC 端。

页面顶部导航栏高亮"我的课程"。

主体以周视图日程表的形式展示本周课程安排。表格横向为星期一到星期五，纵向为节次（第1节到第8节）。
每个课程格子显示：课程名称、上课教室、授课教师。不同课程用不同浅色区分（如淡蓝、淡绿、淡橙）。

周视图下方为"我的课程列表"，以卡片形式展示本学期所有选修/必修课程列表。
每个课程卡片包含：课程封面占位图、课程名称、授课教师、上课时间、学分、课程描述简介。
卡片右下角有"查看详情"按钮。
```

---

## Prompt 4：学生门户 - 校园生活（食堂菜单页）

```
生成一个学生门户网站的食堂菜单浏览页面，PC 端。

页面顶部导航栏高亮"校园生活"。

页面标题"今日食堂菜单"，下方可切换食堂（下拉选择：第一食堂、第二食堂、第三食堂）和选择日期。

菜品按餐段分为 4 个横向 Tab：早餐、午餐、晚餐、夜宵。
当前餐段下方以网格展示菜品卡片，每行 4 个。
每个菜品卡片显示：菜品图片占位、菜名、价格、好评标签（如"热门"推荐"）。
卡片右下角显示点赞数。

页面色调偏暖色（橙黄系），营造食欲感。
```

---

## Prompt 5：学生门户 - 个人中心页

```
生成一个学生门户的个人中心页面，PC 端。

页面顶部导航栏高亮"个人中心"。

左侧为个人资料卡片：学生头像（大图圆形占位）、姓名、学号、班级、专业。下方有"编辑资料"按钮。

右侧上方为账号安全区域：修改密码、绑定手机、绑定邮箱，以列表形式展示，每项右侧显示当前状态（已绑定 / 未绑定）。

右侧下方为系统设置区域：消息通知开关、深色模式开关、语言选择。

```

---

> 使用建议：
> - 先跑 Prompt 1 确定首页整体风格
> - 再依次跑 Prompt 2-5 完善各子页面
> - 不满意直接在墨刀里拖拽修改，或微调 prompt 重新生成
